Frontend :- React and tailwind css


Backend:- Node.js and MongoDB Atlast for database

 Project has 6 module:-

 1) Admin module:- Admin can manage doctors, nurses, beds, ambulance, equipment.
 2) User module:- User can resister and login the system securely as all the credentials are stored as hashes.
 3) book appointment module:- Users can book appointment at particuler time and date.
 4) Order medicine module:- users can order the medicine easily.
 5) feedback module:- Users can provide the feedback according to there experince
 6) Hospital information module:- Users can view the hospital information without login like beds avialble, oxygen avialble, nurses, doctors, etc.

The project is deployed on netlify cloud:- https://serene-brioche-e56c03.netlify.app/

To run the project:-

backend:- node server.js


frontend:- npm run dev
